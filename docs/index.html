CraftBox is a RaspberryPi powered Local Area Network Minecraft Server intended for parents who don't want their kids playing on public servers until they are old enough. The reasons might vary, but we believe parents want to allow their children to play Minecraft, it's just that public servers could be full of Griefers or people who swear or insult. This isn't the environment that Notch envisioned.

CraftBox's features:

  • Optimised Spigot server
  • Some included plugins (NoSpawnChunks, and we're building a time restriction one)
  • Material web management console
  • Creating new worlds
  • Uploading or backing up worlds
  • World configuration (at creation)
  • Server configuration
  • CraftBox configuration (WiFi SSID, console login credentials)
  • Setting time restrictions
  • RCON access to server
  • Easy server restarting process
